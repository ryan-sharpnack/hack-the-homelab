# Journal

Day-by-day build and operations log for Ironbridge Health Alliance.

## Naming convention

`YYYY-MM-DD-day-N.md`

Day count starts at Day 1 for this project. Example: `2026-07-06-day-1.md`

## What goes in an entry

- What was worked on and why
- Commands run / configuration changed
- Problems hit and how they were resolved
- Links to any runbook created or updated as a result
- Anything relevant to the risk register (document 06) — note if a session moves an `IHA-00x` item toward mitigated

Entries here are the raw, dated record. Anything worth reusing gets promoted into `/runbooks` as a clean, repeatable procedure.
